Perguntas Frequentes

Como sugerir um novo ambiente?

Você pode enviar sua sugestão enviando uma mensagem através do formulário acima. Indique qual ambiente gostaria de ver incluído na plataforma.

Posso usar as imagens do tour virtual?

As imagens estão disponíveis para fins educacionais. Para outros usos, entre em contato conosco para discutir as possibilidades.

O projeto é de código aberto?

Sim! O código-fonte estará disponível no GitHub sob licença apropriada. Visite o repositório para contribuir ou adaptar o projeto.

Como funciona o suporte técnico?

Para dúvidas técnicas, envie uma mensagem através do formulário. Responderemos com informações ou soluções para seu problema.
