Projeto ImmersIF Ceres

Desenvolvimento e Implementação de Tecnologias Imersivas para Divulgação Científica e Educacional

Apresentação

O projeto ImmersIF Ceres é uma iniciativa desenvolvida no âmbito do Programa Institucional de Bolsas de Iniciação em Desenvolvimento Tecnológico e Inovação (PIBITI), implementado pelo IF Goiano – Campus Ceres.

Sua missão é democratizar o acesso à educação científica e tecnológica utilizando as mais modernas tecnologias imersivas, especialmente tours virtuais em 360° e realidade virtual, como ferramentas de divulgação científica.

Objetivos

O que buscamos alcançar com este projeto

🎯 Objetivo Geral

Criar e disponibilizar uma plataforma aberta de tours virtuais 360° e ambientes imersivos que permitam a exploração virtual dos espaços científicos, educacionais e institucionais do IF Goiano – Campus Ceres, ampliando o acesso à informação e experiências educacionais para além das barreiras geográficas e temporais.

📚 Objetivos Específicos

  • Mapear, fotografar e processar ambientes selecionados do campus em 360°
  • Desenvolver interface responsiva e acessível para visualização dos tours
  • Implementar sistema modular extensível para futuras adições de ambientes
  • Criar documentação técnica para manutenção e expansão
  • Avaliar impacto educacional da plataforma junto a comunidade
  • Explorar tecnologias de integração com dispositivos VR (Meta Quest, etc.)

Justificativa

Por que este projeto é importante

🌍 Acessibilidade e Inclusão

Muitos interessados em conhecer o IF Goiano enfrentam barreiras geográficas, financeiras ou de mobilidade. Uma plataforma de tours virtuais permite que qualquer pessoa, em qualquer lugar do mundo, possa explorar nossos ambientes de forma imersiva e gratuitamente.

📱 Alinhamento com Tendências Tecnológicas

Realidade virtual e imersivas são tecnologias em crescimento exponencial. Integrar-se a estas tendências posiciona a instituição como inovadora e tecnologicamente atualizada, atraindo alunos e investimentos em pesquisa.

🎓 Educação Científica Engajante

Estudantes aprendem melhor através de experiências imersivas. Tours virtualmente permitem que alunos do ensino básico ou interessados explorem laboratórios e espaços científicos em detalhes, gerando interesse por carreiras em STEM.

💡 Desenvolvimento Tecnológico Local

O projeto é uma oportunidade de formar profissionais em tecnologias imersivas, desenvolvimento web moderno e gestão de projetos. Gera conhecimento interno e contribui para o desenvolvimento tecnológico local.

🏆 Pesquisa e Inovação

Este projeto combina desenvolvimento tecnológico com avaliação do impacto educacional, gerando oportunidades de pesquisa e publicações científicas sobre o uso de tecnologias imersivas em educação.

Etapa Atual

✅ Fase 1: Fundação Web (Atual)

Atualmente, o projeto encontra-se na primeira fase de desenvolvimento, focando na criação de uma plataforma web responsiva e modular como base para futuras expansões.

  • Site institucional com HTML5, CSS3 e JavaScript
  • Sistema visualizador 360° customizado de ambientes com A-frame
  • Estrutura preparada para navegação em VR

Desenvolvimento Autoral (Futuro)

🛠️ Visualizador 360° para navegação em VR independente

Desenvolvimento de um sistema de navegação em VR baseado em A-Frame (WebXR), oferecendo:

  • Controle total sobre a experiência visual e de interação
  • Compatibilidade nativa com dispositivos VR (Meta Quest, HTC Vive)
  • Sistema de hotspots customizável
  • Sem dependência de serviços terceirizados
  • Código aberto e extensível

🔧 Backend Administrativo

Desenvolvimento de sistema administrativo para:

  • Gerenciamento de ambientes (upload de imagens 360°)
  • Criação e edição de hotspots
  • Autenticação de usuários
  • Análise de dados de uso (analytics)

📊 Pesquisa e Avaliação

Integração de ferramentas de pesquisa para avaliar impacto educacional, gerando dados para publicações científicas.

Justificativa Técnica para Desenvolvimento Autoral

Por que não usar plataformas prontas?

Plataformas prontas, apesar de oferecerem soluções rápidas, apresentam limitações críticas para um projeto acadêmico:

  • Falta de controle: Impossibilidade de ajustar interface para necessidades específicas
  • Custos escaláveis: Modelos freemium acumulam custos com crescimento
  • Dependência de terceiros: Risco de descontinuação ou mudanças de política
  • Sem dados para pesquisa: Limitações em coleta de métricas educacionais
  • Oportunidade pedagógica: Desenvolvimento de competências técnicas reais

Vantagens do desenvolvimento próprio

  • Propriedade intelectual: Código aberto e registro de software
  • Escalabilidade: Infraestrutura customizável conforme crescimento
  • Integração acadêmica: Possibilidade de integrar ferramentas de pesquisa
  • Continuidade: Garantia de funcionamento independente de terceiros
  • Potencial de transferência: Replicável em outras instituições
